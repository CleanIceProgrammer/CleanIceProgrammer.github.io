Python 是什么

Python，诞生于 1991 年，由荷兰人 Guido van Rossum 创造，是目前最火的编程语言（很多次都是 TIOBE 排行榜首）。

那么 Python 为何会如此有名？

  1. 简单易学。这是它最直接的原因。Python 简单到令人出乎意料，下到 6 岁上到 60 岁，都可以无门槛无压力轻松地在一个月之内掌握 Python。Python 的简单也使得用它开发项目的速度变得飞快，同时也使它的用于不局限于专业的程序员，会计、老师、或者是摄影师等等，都可以使用 Python 来使它们的工作更加简单。也有大部分的学生掌握 Python，用它写一些好玩的小程序等等。总之，Python 的简单只有在你尝试之后才知道。
  2. 功能强大。Python 不仅简单，功能还十分强大，几乎啥都可以做，包括互联网、物联网、大数据、人工智能、游戏、自动化脚本、编译器、解释器、算法于数据结构等等，你想让它做什么它就做什么。其中，人工智能、大数据、互联网是 Python 的拿手好戏，Python 也因此被称为“专门为人工智能诞生的语言”。
  3. 扩展性强。Python 在各个领域、各个功能都有免费的、庞大的第三方库的支持，使得你可以零门槛使用某些功能，如语言和文字互相转换，图形窗口，网络爬虫等等，都可以使用第三方库简单解决。此外，Python 还被称之为“胶水语言”，它能够于多个其他的语言环境或编程语言想结合，最常见的就是 C/C++，由很多第三方库就是使用 C/C++ 编写的，在导入时能够直接和 Python 代码结合。此外，Python 的解释器就是使用 C 写的呢！
  4. 免费且开源。免费的 Python 让每个人都可以体验到编程的乐趣。此外，Python 还将它用 C 语言写的解释器完全开源，你查看它的源代码，学习里面用到的技术和技巧，也可以修改它，创造一门与 Python 很相像的自己的语言。
  5. 跨平台。Python 允许在多个平台运行，包括常见的 Windows，Linux，MacOS 等等，也包括 Android，iOS 等等移动端操作系统，这使得一份 Python 代码几乎可以不做任何修改就在其他平台运行，具有极强的跨平台性。

上面便是 Python 的有点，当然肯定不止这些。等你学了 Python，自然能够领略它的优点。

Python 编程语言的类型介绍

Python 是一门多范式动态强类型解释语言：

多范式：Python 支持多种编程范式，包括面向对象、过程式、函数式。其中以面向对象为主。

动态类型：Python 是动态类型语言，这意味着你有更大的编程空间，有更大的编程自由度，你不必在源码中标注类型，也可以在运行时改变类型。

强类型：这使得 Python 的数据操作变得安全，代码更容易维护。

解释型：这让 Python 拥有更多的自由度和跨平台性，但是因此变得稍微有些慢。
